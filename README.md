# Tip & Discount Helper App

A simple Flutter application that helps users quickly calculate discounts, tips, and final prices.  
This project was developed as part of a mobile development course and demonstrates UI design, navigation, and simple state management in Flutter.

  - Price after discount
  - Tip amount
  - Final price

- Clean and simple 2-page interface:
  - **Home Page**
